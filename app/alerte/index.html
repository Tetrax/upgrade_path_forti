Nouvelle alerte

Déclarer une alerte interne

Mise en forme : **gras**, __souligné__, une ligne commençant par "- " pour une puce, ligne vide pour un nouveau paragraphe. Coller (Ctrl+V) ou glisser une image directement dans le champ, ou utiliser le bouton Image.

Cocher une ou plusieurs versions. L'alerte s'affichera pour tout chemin d'upgrade qui passe par une de ces versions.

Cocher un point de départ par branche concernée (ex: 7.2.11 et 7.4.5 et 7.6.1 pour un correctif backporté dans ces trois branches). L'alerte s'affiche pour cette version et les versions suivantes de la même branche (7.4.5 déclenche sur 7.4.5+, mais pas sur 7.2.x ni 7.6.x sauf si cochées aussi).

Pour retrouver facilement le bug dans les release notes Fortinet plus tard (Resolved/Known issues).
